I. Dispositio Volumetrica — 3D Staggered Arrangement
Isometric projection · each disk at unique (Y, Z) position · observer at left
Three disks at genuinely different (Y,Z) positions — each at unique height and depth. From the observer's eye 80–120 cm away, all three overlap in the visual field simultaneously. The depth stagger produces physical parallax: as the observer moves laterally, the arm patterns shift relative to each other, making the interference field feel alive in three dimensions.
II. Brachium Extensibile — Arm Extension Mechanism
Exploded detail · rack-and-pinion servo drive · SG90 per arm · PCA9685 I²C on hub
Each arm is a two-part aluminium extrusion: fixed outer tube (hub side) and sliding inner rod. An SG90 micro-servo at the hub drives a rack-and-pinion gear on the inner rod. All servos on one disk connect to a PCA9685 16-channel driver board mounted on the rotating hub — controlled via I²C through the existing slip-ring SDA/SCL channels. Zero new wiring required.
III. Lexicon Formarum — Pattern Vocabulary by Arm Extension State
Front view · eight configurations from arm extension combinations · 4-arm disk (D1) shown · additional complexity from multi-disk overlay
Eight configurations from the extension parameter space. ① Standard cross — all arms at maximum. ② Compact — all retracted. ③ Diamond — orthogonal axes at different lengths. ④ Breathing — all arms oscillating in sync with music BPM. ⑤ Implicit spiral — arms in progressive extension; the spiral rotates as the disk spins. ⑥ Iris — all arms closing in unison. ⑦ RGB colour mix — three disk arm trails crossing, additive colour composition in the observer's visual cortex. ⑧ Lissajous lock — integer RPM ratios (2:3:5) cause the three-disk system to form stable standing figures.
IV. Systema Chromaticum — RGB Disk Colour Assignment
Additive colour mixing in visual field · persistence-of-vision colour space · front view
Assigning red, green, and blue (or any three complementary hues) to the three disks turns the visual overlap zone into a persistence-of-vision additive colour space. As disks rotate at different speeds, the overlap geometry changes continuously — the colour composition is never the same twice. The RGBW capability of SK6812 allows any hue per pixel, making the colour assignment fully scriptable from the VJ API.
V. Lingua Machinae — Extended VJ/DJ Script API
JSON keyframe schema · new arm_extension[] parameter · all fields scriptable
The existing JSON preset format gains two new parameters per disk: disk_color (full RGB hex, overrides per-pixel LED colour) and disk_arm_ext[] (array of 0.0–1.0 floats, one per arm). Both interpolate between keyframes using the specified curve. The PCA9685 receives arm extension targets via I²C through the slip ring; the ESP32 already interpolates motor RPM between keyframes — the same interpolation loop handles arm extension.
VI. Nexus Extensus — Updated Control Architecture
ESP32 + PCA9685 per disk · I²C chain through existing slip rings · no new wiring
The existing control architecture requires only three PCA9685 servo driver boards (one per disk hub) and micro-servos on each arm. All communicate via the I²C bus already present in the slip ring. Total additional cost approximately €60–105 across all three disks. Firmware addition is minimal — an arm-extension interpolation loop parallel to the existing RPM PID loop.
On the staggered plane arrangement: the depth offsets (Z = 0, −18 cm, −36 cm) are chosen such that at 100 cm viewing distance, all three disk diameters (60, 45, 30 cm) subtend overlapping arcs in the observer's 45° visual field. The physical moiré between rotating skeletal arms at different depths is a fundamentally different phenomenon from persistence-of-vision moiré — it involves genuine geometric interference between three-dimensional structures, producing a volumetric light field rather than a flat pattern.
On extendable arms and the notion of a display: with 4+3+2=9 independently-addressable radial positions per disk, plus the depth and height stagger, the machine effectively becomes a low-resolution cylindrical persistence-of-vision display. Each arm tip traces an arc whose radius is controllable. The composer of a VJ script is not merely setting parameters — they are sculpting a time-varying three-dimensional luminous form, lit from within, responsive to the kinetics of the music.
On additive colour: that the visual cortex performs additive colour mixing of overlapping motion trails is the same mechanism by which a television produces colour from three phosphors. The difference is that here the pixels are arcs of motion, the frame rate is the rotation speed, and the canvas is the inside of the observer's skull. The machine does not project images onto the participant — it manufactures them within.
